# Aura Wellness - Premium Health & Nutrition

A modern, responsive e-commerce storefront for a health and wellness brand. Built with React and Vite.

## Features

- **Product Catalog**: Beautifully presented products with categories, descriptions, and pricing.
- **Interactive Shopping Cart**: Add/remove items and adjust quantities with a real-time sidebar.
- **Premium Design**: Clean, minimalist aesthetic inspired by modern DTC wellness brands.
- **Responsive Layout**: Optimized for all device sizes.

## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

### Development

Run the development server:
```bash
npm run dev
```

### Build

Create a production build:
```bash
npm run build
```
